## Development



Setting up:
```bash
git clone git@github.com:jaid/browserslist-config-jaid-node.git
cd browserslist-config-jaid-node
npm install
```
Testing:
```bash
npm run test:dev
```
Testing in production environment:
```bash
npm run test
```
